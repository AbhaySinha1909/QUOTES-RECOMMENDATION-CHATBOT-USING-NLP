About QuoteGenie

An AI-powered Quotes Recommendation Chatbot built using Rasa NLP framework.

✨ Features

  • Motivational Quotes
  • Inspirational Quotes
  • Love Quotes
  • Funny Quotes
  • Success Quotes
  • Real-time Web Chat Interface
  • Automated Intent Recognition

⚙ How It Works

QuoteGenie uses Rasa’s DIETClassifier for intent classification and TEDPolicy for dialogue management. When a user sends a message, the system identifies the intent and returns a relevant quote from the trained dataset.

🚀 Technology Stack

  • Rasa 3.6 (Backend AI Engine)
  • Python 3.10
  • HTML, CSS, JavaScript
  • REST API Integration
